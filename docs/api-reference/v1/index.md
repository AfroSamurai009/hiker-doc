# REST API v1

Mobile API endpoints for anonymous Instagram data retrieval.

All endpoints use `GET` method and require the `x-access-key` header. See [Authentication](../../getting-started/authentication.md).

| Resource | Description | Endpoints |
|----------|-------------|-----------|
| [User](user.md) | Profiles, followers, following | 21 |
| [Media](media.md) | Posts, reels, likes, comments | 16 |
| [Stories](stories.md) | User stories | 7 |
| [Highlights](highlights.md) | Story highlights | 4 |
| [Hashtags](hashtags.md) | Hashtag info and media | 8 |
| [Locations](locations.md) | Location info and media | 7 |
| [Search](search.md) | Search users, hashtags, places | 9 |
