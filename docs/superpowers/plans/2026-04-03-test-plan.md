## 2. Код-примеры — выполнить с реальным токеном

Заменить `YOUR_TOKEN` на боевой токен. Каждый пример должен вернуть валидный JSON.

### Quick Start

- [ ] curl — `GET /v1/user/by/username?username=ronaldo` → 200, JSON с полями pk, username, follower_count
- [ ] Python (hikerapi SDK) — `cl.user_by_username_v1("ronaldo")` → dict с pk
- [ ] Python (requests) — то же через requests.get → 200
- [ ] JavaScript (fetch) — то же через fetch → JSON
- [ ] Pagination curl — первая страница followers возвращает данные + end_cursor
- [ ] CSV export — скрипт создаёт файл users.csv
